# Example Prompts for AI System Architect

## E-Commerce Projects

### Basic E-Commerce Platform
```
Build an e-commerce platform with product catalog, shopping cart, payment processing, 
and order management. Include features for sellers to list products and buyers to 
search and purchase. Should be scalable for thousands of products.
```

### Marketplace Platform
```
Create a marketplace like eBay or Shopify where multiple vendors can sell products. 
Include vendor management, commission system, review/rating system, and dispute resolution. 
Consider international payments and shipping integration.
```

## Social & Communication

Build a real-time chat application with features like direct messaging, group chats,

## Tips for Best Results

1. **Be Specific**: Include your target audience, core features, and constraints
2. **Mention Scale**: How many users/records are you expecting?
3. **Tech Preferences**: Any specific technologies you want to use or avoid?
4. **Timeline**: When do you need this ready?
5. **Budget**: Any budget constraints (startup, enterprise, etc.)?
6. **Integration**: Any third-party services you need to integrate with?

## Example with Full Context

```
Build a modern e-commerce platform for selling handmade crafts. 
Target: Small artisans and craft makers (1,000+ sellers, 50,000+ potential customers)
Core Features:
- Product listing and management
- Shopping cart and checkout
- Payment processing with Stripe
- Order management and tracking
- Customer reviews and ratings
- Search and filtering

Tech Preferences:
- React for frontend
- Node.js/Python for backend
- PostgreSQL for database
- AWS for hosting

Timeline: 3 months MVP (Minimum Viable Product)
Budget: Startup budget ($10-20K)
Must integrate with: Stripe payments, SendGrid for emails

Additional: Need vendor dashboard for sellers and admin panel for content moderation
```

This detailed context helps generate more targeted and practical blueprints!
